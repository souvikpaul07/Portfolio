# Personal Portfolio

A modern developer portfolio, showcasing my experience in backend development, microservices architecture, and application modernization.

## 🔗 Live Demo
https://paul-souvik-portfolio.vercel.app/


## 🚀 Overview

This portfolio highlights my work as a software developer, including projects, technical skills, work experience, and certifications. It is designed with a clean, responsive UI to provide a seamless user experience across devices.

## 🛠 Tech Stack

* Next.js
* React.js
* TypeScript
* JavaScript
* HTML/CSS

## 📌 Features

* Structured project showcase with detailed descriptions and tech stacks
* Technical expertise categorized by domain
* Professional work experience section
* Certifications and credentials
* Fully responsive design

## ⚙️ Getting Started

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## 📂 Project Structure

* `app/` – Main application pages
* `components/` – Reusable UI components
* `data/` – Static portfolio data (projects, skills, etc.)

## 📦 Deployment

The project is deployed using Vercel for fast and reliable hosting.
